Soru Sor

API anahtarını bir daha sormaması için:

  1. key.example.js dosyasını key.js olarak kopyala.
  2. "sk-...." kısmını kendi OpenAI anahtarınla değiştir.
  3. Sayfayı yenile; anahtar otomatik olarak alınacak ve giriş kutusu kilitlenecek.

GitHub web’de düzenlemek için: Depoda key.example.js dosyasını açıp kalem (Edit) ikonuna tıkla, anahtarını ekleyip dosyayı key.js adıyla kaydet; ardından sayfayı yenile.

Bağlantıdan otomatik eklemek için: Adresin sonuna ?key=... veya #key=... ekle. Sayfa açılınca anahtar sekmede saklanır, adres çubuğu temizlenir ve kutu kilitlenir.

Anahtarı paylaşma ve key.js dosyasını depoya ekleme.
