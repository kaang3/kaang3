# MINANCE Front-End

Bu projedeki dosyalardan birini tek seferde kopyalamak istersen aşağıdaki yöntemleri kullanabilirsin:

1. **Raw görünümü:** 
   - `script.js` dosyasını GitHub üzerinde aç.
   - Sağ üstteki **Raw** düğmesine tıkla.
   - Sayfa düz metin olarak açıldığında `Ctrl+A` (veya `Cmd+A`) ile tamamını seçip `Ctrl+C` ile kopyala.

2. **Doğrudan indirme:**
   - Dosya sayfasının üstündeki üç nokta menüsünden **Download file** seçeneğini kullanarak tek dosyayı indir.

3. **Depoyu indir / klonla:**
   - Tüm projeyi bilgisayarına almak için `Code` → `Download ZIP` yolunu veya `git clone` komutunu kullan.

Bu yöntemler dosyaları parça parça seçmek yerine tek hamlede kopyalamanı sağlar.
