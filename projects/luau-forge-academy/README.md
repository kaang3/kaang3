# Luau Forge Academy

Roblox Luau öğrenmek için Duolingo tarzı, parlak ama olgun görünümlü tek dosyalık web uygulaması.

## İçerik

- 78 derslik Luau rotası
- Yüzeysel `print`, `local`, `if` konularından başlayıp `RemoteEvent`, `ModuleScript`, `DataStore`, `pcall`, `metatable` ve state machine konularına ilerler
- Quiz, kod alıştırması, XP, seri, ilerleme kaydı
- Kod Dojo ve cep kütüphanesi
- PNG veya harici asset gerektirmez; `index.html` tek başına açılır

## Kullanım

`projects/luau-forge-academy/index.html` dosyasını tarayıcıda aç.

Bu PR mevcut dosyaları silmez veya değiştirmez; sadece `projects/luau-forge-academy/` altında yeni dosyalar ekler.
